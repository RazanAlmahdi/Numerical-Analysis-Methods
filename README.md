# NumericalAnalysis

## Purpose
<!--Purpose of the project-->
This poject is an implementation of multiple numerical methods in Matlab.
<!--Header 2 description of the project-->
## Description

<p style="text-align: justify">
The methods include:
  
* Euler Method
* Secant Method
* Taylor Method
* Bisection Method
* Newton Method
* Rungue Kutta Method
